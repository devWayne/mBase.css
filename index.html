Overview

mBase基于less开发，为解决在移动端垂直频道开发中，样式不统一，重复，冗余等问题，同时以提升开发效率为目标，不断自我完善中。

介绍-Introduce

mBase使用less开发，引入的基础库包括normalize-css,使用postcss进行打包，前缀修正。

发布有2个版本mBase.css和mBase.lite.css,mBase.css支持所有的功能和样式，mBase.lite.css在mBase.css进行删减，除去了不常用的样式组件，来减小文件大小，提高性能。

引入-Usage

1.全局方式引入:

mb.css和mb.lite.css的文件可以在项目源码中的dist文件夹下找到

<link href="css/mb.css" rel="stylesheet" type="text/css">

或者

<link href="css/mb.lite.css" rel="stylesheet" type="text/css">

2.通过Bower引入:

cd yourProject
bower install mBase

或者

cd yourProject
bower install mBase.lite

3.通过Cortex引入:

cd yourProject
cortex install --save mBase

或者

cd yourProject
cortex install --save mBase.lite
